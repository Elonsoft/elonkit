## Constants

Injection token that can be used to configure the default options for all components within an app.

```ts
import { ES_TABLE_ACTIONS_DEFAULT_OPTIONS } from '@elonsoft/elonkit/ui/table-actions';

@NgModule({
  providers: [
    {
      provide: ES_TABLE_ACTIONS_DEFAULT_OPTIONS,
      useValue: {
        total: 10,
      }
    }
  ]
})
```
