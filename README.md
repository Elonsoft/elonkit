# Elonkit

## Installation

```bash
yarn add @elonsoft/elonkit angular2-text-mask text-mask-addons
```

Add library assets to your `angular.json` config:

```json
"assets": [
  "src/favicon.ico",
  "src/assets",
  {
    "glob": "**/*",
    "input": "./node_modules/@elonsoft/elonkit/assets",
    "output": "./assets"
  }
],
```
